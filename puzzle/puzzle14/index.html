Puzzle 14: Schiebepuzzle

Stellen Sie durch horizontales und vertikales Verschieben der einzelnen Bausteine (Zuerst Checkbox der Richtung wählen, dann Click auf Baustein) die Vorlage her!
Der Baustein bewegt sich so lange, bis er auf den nächsten Stein oder einen Farbblock trifft. Stösst er auf einen Farbblock, so nimmt er dessen Farbe an!

Ihr Browser kann kein Canvas!
Vorlage:

Ihren jeweils letzten Zug können Sie durch Klicken auf 'Zug -' rückgängig machen.
Klick auf 'Clear' stellt Anfangszustand her.

Verschieben des Bausteins nach:
links      rechts      oben      unten
