Jassen: Simulation der Wahrscheinlichkeiten von Dreiblatt, Vierblatt usw.

Beim Jassen werden einem Spieler 9 von 36 Karten verteilt.
Hier können Sie simulieren, mit welcher Wahrscheinlichkeit ein Spieler genau ein Dreiblatt, genau ein Vierblatt... erhält.
Wählen Sie n = 1, so werden Ihnen die gezogenen Karten und ev. Dreiblätter usw. angezeigt.
Für n > 1 werden die Anzahl der Dreiblätter usw. angezeigt. Falls n ≥ 100, so wird auch die relative Häufigkeit und damit die Wahrscheinlichkeit approximativ berechnet.
