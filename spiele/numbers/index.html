Zahlenspiel

Dieses Spiel simuliert den einen Teil der französischen Fernsehshow "Les Chiffres et Les Lettres".
Das Problem besteht darin, aus 6 gegebenen Zahlen (zwischen 1 und 100) eine Zielzahl (zwischen 101 und 1000) zu berechnen. Dabei dürfen die Grundrechenarten verwendet werden, aber jede Zahl höchstens einmal.
Das Programm kann selbst Probleme lösen, oder Sie können die Probleme lösen.

Wenn Sie eine Zahl in ein Inputfeld eingegeben haben, so müssen Sie auf 'Reset' klicken.
'Reset' löscht auch Lösungsversuche. Nach Klick auf 'Solve' sucht der Computer eine Lösung.
