Infos zum Programm Squiggly Sudoku

  • Ein Feld kann durch Mausklick oder mit den Cursortasten aktiviert werden (Aktuelles Feld hat hellblauen Hintergrund)
  • Die Eingabe der Ziffern erfolgt mit den Buttons oder mit Hilfe der Tastatur (delete-Taste löscht die Ziffer).
    Das Programm lässt für die Eingabe die - gemäss den Regeln - in den Leerfeldern noch möglichen Ziffern zu. Es ist aber wegen der Eindeutigkeit der Lösung jeweils natürlich nur genau eine dieser Ziffern richtig!
  • Richtig?-Button: Erscheint, sobald Sie Ziffern zu einem Lösungsversuch eingeben. Zeigt, ob Ihr Versuch noch korrekt ist.
  • Aktivierung der Checkbox 'Verfügbare Ziffern im akt. Feld' zeigt die verfügbaren Ziffern im aktuellen Feld.
    Aktivierung der Checkbox 'Nicht mögliche Felder für Ziffer':
    Ziffern 1 bis 6 (bzw.9): die nicht mehr möglichen Felder für die gewählte Ziffer sind hellgrün, die möglichen haben die Farbe der entsprechenden Box.
  • Save-Button: speichert den aktuellen Zwischenstand.
  • Load-Button: ladet den vorher mit Save gespeicherten Zwischenstand.
  • Solve-Button: versucht, eine Lösung der Stellung vom aktuellen Brett zu finden. Wenn Sie 'Solve' wählen, so wird Ihr Lösungsversuch abgebrochen. Wählen Sie also zur Kontrolle Ihrer Lösungsversuche den Button 'Richtig?' und nicht 'Solve'.
  • Auswahlfeld 'Prob': vorgespeicherte Probleme p1 bis p9 (p8 ist ein Problem mit originaler Boxstruktur).
  • Standard Probleme in Probneu laden ladet 9 Probleme mit originaler Boxstruktur in 'Probneu', darunter auch ein Minimal-Problem mit 17 Zeichen.
  • Create-Button: erzeugt ein neues, eindeutig zu lösendes Problem. Bitte etwas Geduld! Eventuell kann der Computer in vernünftiger Zeit kein Problem erzeugen. Klicken Sie dann einfach nochmals auf 'Create'.
    Falls ein neues Problem erzeugt wird, so wird die Problemstellung gespeichert und Sie können sie mit Load wieder laden - solange Sie sie nicht mit Save überschrieben haben. Sie können sie aber auch in 'Probneu' speichern.
  • Clear-Button: erzeugt leeres Startfeld mit der aktuellen Boxstruktur.
  • NeueBoxstruktur Comp lässt den Computer eine Zufallsstruktur (zusammenhängende Boxen) erzeugen. Bei Situation 9x9 brauchen Sie dann bei der Erzeugung einer eindeutigen Problemstellung (mit Create) meistens viel Geduld. Es lohnt sich aber, mehrere Versuche mit Create zu starten. Hat der Computer mal eine eindeutige Problemstellung gefunden, so findet er die nächste eindeutige Problemstellung derselben Boxstruktur leichter.
  • Durch Klick auf NeueBoxstruktur User können Sie eine eigene Struktur schaffen. Die letzte Box (übriggebliebene 6 bzw 9 Felder) wird vom Computer hinzugefügt.
    Klick auf Create erzeugt dann ein neues Problem. Falls Sie mehrmals hintereinander die Meldung 'Create nicht gelungen' erhalten, so gibt es eventuell für Ihre erzeugte Boxstruktur kein mögliches eindeutiges Problem. Erzeugen Sie dann eine neue Struktur oder klicken Sie auf 'Clear', um eine vordefinierte Struktur zu laden.
  • Das erste durch Create entstandene Problem nach jeweils neuer Boxstruktur wird automatisch gespeichert und ist via 'Probneu' abrufbar. Mit Save aktuelles Problem als Probneu wird die aktuelle Problemstellung zu den Problemen in 'Probneu' hinzugefügt.
    Insgesamt können für 6x6 und 9x9 je 9 neue Probleme gespeichert werden.
    Sie können diese gespeicherten Probleme mit Download der neuen Probleme als Textdatei (NeueProbleme.txt) auf der Harddisk sichern. Je nach Ihren gewählten Einstellungen im Browser können Sie den Speicherort auswählen oder Sie finden die Datei im Ordner 'Downloads' auf Ihrem Compi.
    Beim upload der Textdatei werden ev. vorhandene Probleme in 'Probneu' gelöscht. Sie sollten sie also vor dem Laden mit NeueProbleme(1).txt usw. sichern.
  • Nach Klick auf CreateUser können Sie die Ziffern für eine neue Problemstellung selbst eingeben. Mit Test:Eindeutig? prüfen Sie die Stellung auf Eindeutigkeit. Bei der Situation 9x9 und geringer Anzahl Ziffern kann diese Prüfung sehr lange dauern. Wenn die Stellung eindeutig ist, erscheint der Button 'Save aktuelles Problem ...' und Sie können sie unter 'Probneu' speichern.";
