Infos zum Programm Squiggly Sudoku (v3.3.0) auf www.mathematik.ch

  • Vorgewählt ist Situation 6x6. Unterste Checkbox angekreuzt wählt Situation 9x9.
  • Ein Feld kann durch Mausklick oder mit den Cursortasten aktiviert werden (Aktuelles Feld hat hellblauen Hintergrund)
  • Die Eingabe der Ziffern erfolgt mit den Buttons oder mit Hilfe der Tastatur.
    Das Programm lässt für die Eingabe die - gemäss den Regeln - in den Leerfeldern noch möglichen Ziffern zu. Es ist aber wegen der Eindeutigkeit der Lösung jeweils natürlich nur genau eine dieser Ziffern richtig!
  • Create-Button: erzeugt ein neues, eindeutig zu lösendes Problem. Bitte etwas Geduld! Eventuell kann der Computer in vernünftiger Zeit kein Problem erzeugen. Klicken Sie dann einfach nochmals auf 'Create'.
    Falls Sie ein neues Problem erzeugen, so wird die Problemstellung gespeichert und Sie können sie mit Load wieder laden - solange Sie sie nicht mit Save überschrieben haben.
  • Auswahlfeld 'Prob': ladet gespeicherte Probleme p1 bis p8 (mit p8 laden Sie die originale Sudoku Boxstruktur)
  • Solve-Button: versucht, eine Lösung der Stellung vom aktuellen Brett zu finden.
  • Clear-Button: erzeugt leeres Startfeld (reset) mit der aktuellen Boxstruktur. Drücken Sie 'Clear' während Sie eine neue Boxstruktur erzeugen, so brechen Sie diese Erzeugung ab und laden eine gegebene symmetrische Boxstruktur.
  • Save-Button: speichert den aktuellen Zwischenstand.
  • Load-Button: ladet den vorher mit Save gespeicherten Zwischenstand.
  • Lock-Button: aktuelle Ziffern können nachher nicht mehr verändert werden (rote Ziffern).
  • Back-Button: macht den jeweils letzten Zug rückgängig.
  • Aktivierung der Checkbox 'Verfübare Ziffern' zeigt die verfügbaren Ziffern im aktuellen Feld.
  • Aktivierung der Checkbox 'Mögliche Felder':
    Ziffern 1 bis 6 (bzw.9): die nicht mehr möglichen Felder für die gewählte Ziffer sind hellgrün.
  • Durch Klick auf Neue Boxstruktur Benutzer können Sie eine eigene Struktur schaffen. Die letzte Box (übriggebliebene 6 bzw 9 Felder) wird vom Computer hinzugefügt.
    Klick auf Create erzeugt dann ein neues Problem. Falls Sie mehrmals hintereinander die Meldung 'Create nicht gelungen' erhalten, so gibt es eventuell für Ihre erzeugte Boxstruktur kein mögliches eindeutiges Problem. Erzeugen Sie dann eine neue Struktur oder klicken Sie auf 'Clear', um zum Anfang (mit symm. Boxstruktur) zurückzukehren.
  • Neue Boxstruktur Computer lässt den Computer eine Zufallsstruktur (zusammenhängende Boxen) erzeugen. Bei Situation 9x9 brauchen Sie dann bei der Erzeugung einer eindeutigen Problemstellung (mit Create) meistens viel Geduld. Es lohnt sich aber, mehrere Versuche mit Create zu starten. Hat der Computer mal eine eindeutige Problemstellung gefunden, so findet er die nächste eindeutige Problemstellung derselben Boxstruktur leichter.
  • Sie können auch nach Eingabe von Ziffern beliebige Squiggly lösen oder durch den Computer versuchen lösen zu lassen.
