Suguru

Das Ziel des Spiels ist, in jeder Box (eine Box ist durch fette Linien und Farbe markiert) genau die Ziffern 1 bis N (Anzahl Zellen der Box) zu haben. Dabei dürfen keine benachbarten Zellen dieselbe Ziffer aufweisen.
Die Probleme sind eine Teilmenge der Probleme der Suguru.apk (s. Externe Links) von Aliaksandr Uvarau.
Der Lösungsalgorithmus (kein Backtracking!) ist von Dirk Kok.
Sie können das angezeigte Problem lösen, ein Problem durch den Computer lösen lassen ('Solve', 'Step' zeigt Lösung schrittweise) oder eine Problemstellung neu eingeben ('Eingabe neues Suguru'). Ihre neu erstellten Sugurus können Sie mit 'Download' auf Ihren Computer speichern und später mit 'Upload' wieder raufladen.
Falls Sie bei einer Eingabe einer Ziffer unsicher sind, so klicken Sie auf ✎. Die Eingabe erfolgt dann mit kleinerem Font und Sie können - falls nötig - mit ↶ wieder zu dieser Eingabe zurückkehren.
