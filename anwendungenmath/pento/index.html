Pentominos Feld 10x6

Das Programm zeigt den Algorithmus zum Finden von Lösungen des 10x6 Pentominos-Problems.

Ihr Browser kann kein Canvas!

Anzahl Lösungen: 0
  • Position von Kreuz
  • Verzögerung in Millisekunden
  • Nach Klick auf Go löst der Computer das Puzzle. Klick auf Stop unterbricht diesen Vorgang.
    Der Computer stoppt nach jeder gefundenen Lösung.
  • Klick auf Step lässt den Computer das Puzzle schrittweise lösen.
  • Die Geschwindigkeit können Sie mit den entsprechenden Befehlen steuern.
  • Reset startet mit der aktuellen Position des Kreuzes neu.
  • Da die Anfangsposition des Kreuzes vorgewählt wird, erzeugt das Programm nur nichtkongruente Lösungen. Es gibt insgesamt 2339 solche Lösungen.
