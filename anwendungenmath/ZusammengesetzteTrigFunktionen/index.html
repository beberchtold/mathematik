Graphen von zusammengesetzten Sinus und Cosinusfunktionen

Wählen Sie zuerst mittels Radiobutton (Sinus oder Cosinus), ob der Graph der Funktion mit Gleichung y = a sin[ω(x - h π)] + k oder der Graph der Funktion y = a cos[ω(x - h π)] + k dargestellt werden soll.

Sie können die Werte von a, ω, h und k mit der Maus verändern, nachdem Sie den entsprechenden Radiobutton aktiviert haben:
Verschieben Sie die Maus im Diagramm vertikal (Verändern von a bzw. k).
Verschieben Sie die Maus horizontal (Verändern von ω bzw. h).
Alternativ können Sie diese Werte mit 'Step' auch schrittweise verändern.
Sie können die Werte aber auch direkt in die entsprechenden Felder schreiben.
