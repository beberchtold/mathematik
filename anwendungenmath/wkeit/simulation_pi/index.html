Berechnung von Pi mit MonteCarlo-Methode

Berechnung mit 16000 Punkten.

Ihr Browser kann kein Canvas!


Pi wird mit Startwert 32000 Punkten berechnet. Sie können diese Anzahl verändern: min=8000, max=512000.
Je mehr Punkte Sie wählen, desto besser wird i.a. die Approximation für Pi.

Vergleich: Exakter Wert für Pi = 3.14159265...

Man bestimmt eine vorgewählte Anzahl Zufallspunkte in einem Quadrat der Seitenlänge 1 (die x-Koordinate und die y-Koordinate sind also Zufallszahlen zwischen 0 und 1).
Liegt der Punkt im Viertels-Einheitskreis, so wird er blau gezeichnet, sonst rot.
Das Verhältnis der Anzahl blauen Punkte zur Total Anzahl Punkte ist eine Näherung für Pi/4.
